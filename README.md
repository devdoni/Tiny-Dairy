# 📔 Tiny Dairy

**React와 LocalStorage를 활용한 간단한 일기 작성 서비스**

---

## 🛠️ 주요 기능
- ✅ 회원가입 / 로그인 / 정보 수정  
- ✅ 일기 작성, 수정, 삭제  
- ✅ 작성한 일기 리스트 제공
